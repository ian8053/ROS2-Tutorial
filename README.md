# ROS2 Tutorial

ROS2 Humble 學習筆記與練習程式碼

## 目錄結構

| 資料夾 | 主題 | 說明 |
|--------|------|------|
| `01_nodes_topics_services` | Nodes, Topics, Services | ROS2 核心通訊概念 |
| `02_publisher_subscriber` | Publisher/Subscriber | 撰寫發布者與訂閱者 |
| `03_launch_files` | Launch Files | 批次啟動多個節點 |
| `04_tf2` | TF2 座標轉換 | 機器人座標系統管理 |
| `05_urdf` | URDF 機器人模型 | 描述機器人外觀與關節 |

## 學習順序

1. **01_nodes_topics_services** - 先理解 ROS2 的基本架構
2. **02_publisher_subscriber** - 動手寫第一個 ROS2 程式
3. **03_launch_files** - 學會管理多節點系統
4. **04_tf2** - 座標轉換是機器人系統的核心
5. **05_urdf** - 建立機器人模型

## 環境

- Ubuntu 22.04 LTS
- ROS2 Humble
